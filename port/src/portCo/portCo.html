Logo
Design
“코인 - 주식 - 부동산"의 앞 글자를 딴 “코주부"가 브랜드명으로 코주부하면 떠오르는 코가 큰 안경에서 모티브를 얻어 코주부의 로고를 제작하였습니다. 코인과 다양한 금융정보를 전달하는 것으로 안경에는 각 달러와 비트코인의 기호를 표시하여 직관적으로 표현하였습니다. 각 국문과 영문으로 제작되어 다양하게 사용할 수 있습니다. 메인 컬러인 옐로우와 블루를 사용하여 서로에게 포인트가 되어 로고의 특색을 더 잘 보여줍니다.

Project
Overview

코주부의 반응형 랜딩 페이지와 웹플랫폼 UIUX를 디자인하였습니다. 로고에서 느껴지듯 편하게 다가갈 수 있는 일러스트를 사용하여 쉽게 다가올 수 있도록 하였습니다. 랜딩 페이지에서 플랫폼으로 이동하면 정확하고 신뢰하는 정보를 전달할 수 있도록정갈한 디자인과 선에 맞춘 레이아웃을 사용하여 신뢰감을 주었습니다. 사용자의 편의를 위해 각 페이지별로 동일한 레이아웃을 사용하여 쉽게 사용할 수 있도록 하였습니다.

랜딩 페이지의 경우 피그마에서 프로토타입을 사용하여 실제 사이트와 같이 구현하여 클라이언트와의 소통을 수월하게 하고 개발자와의 협업에서 원하는 방향으로 개발할 수 있도록 하였습니다.
